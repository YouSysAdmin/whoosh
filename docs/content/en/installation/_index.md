## Requirements

- **Operator machine** (where whoosh runs): SSH access to the targets. The Go toolchain is needed only to
  [build from source](/installation/from-source/) or to compose a
  [custom binary with extra plugins](/installation/custom-plugins/).
- **Target hosts**: only `git`, `tar`, and a POSIX shell - no agent or daemon is installed.
